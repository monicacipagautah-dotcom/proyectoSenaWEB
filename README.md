# Proyecto SENA Web

## Descripción

Proyecto web desarrollado en Java como parte del proceso formativo SENA.  
La aplicación permite gestionar usuarios, clientes, proveedores y servicios mediante operaciones CRUD conectadas a una base de datos MySQL usando JDBC.

## Módulo Desarrollado

El módulo implementado corresponde a la gestión principal del sistema de servicios:

- Registro e inicio de sesión de usuarios.
- Registro de usuarios con rol Cliente o Prestador.
- Gestión de clientes.
- Gestión de proveedores.
- Gestión de servicios.
- Conexión a base de datos mediante JDBC.

## Funcionalidades

### Usuarios

- Registrar usuarios.
- Consultar usuarios registrados.
- Actualizar información de usuarios.
- Eliminar usuarios.
- Iniciar sesión.

### Clientes

- Registrar clientes asociados a usuarios con rol Cliente.
- Consultar clientes registrados.
- Actualizar información de clientes.
- Eliminar clientes.

### Proveedores

- Registrar proveedores asociados a usuarios con rol Prestador.
- Consultar proveedores registrados.
- Actualizar información de proveedores.
- Eliminar proveedores.

### Servicios

- Registrar servicios.
- Consultar servicios registrados.
- Actualizar servicios.
- Eliminar servicios.
- Asociar servicios con clientes y proveedores existentes.
